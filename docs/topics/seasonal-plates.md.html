Disabling Seasonal Plates

Seasonal Plates are a feature where certain months have different styles of plates. However, disabling this is very easy.

Open Misc Config

Grab out the llplates/confmisc.lua file, and find the PLATE.Months section.

PLATE.Months = {}
PLATE.Months["03"] = "spring.png"
PLATE.Months["04"] = PLATE.Months["03"]
PLATE.Months["05"] = PLATE.Months["03"]

From here, you can change the name of individual plates, so if you wanted to change per month, you could do that.

Disabling All Seasons

However, if you want to just disable all seasonal plates, you remove all the PLATE.Months section, except for the first line (PLATE.Months = {})

Finally

Save the config, and re-upload. Bish bash bosh, all done.
